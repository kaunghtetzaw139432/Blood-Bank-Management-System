# 🏥 Blood Bank Management System

A robust Desktop Application built with **C# (.NET Framework)** and **SQL Server** to manage donor records and blood inventory efficiently.

## 🌟 Key Features
* **Donor Management:** Add, Update, Search, and Delete donor records.
* **Stock Tracking:** Real-time monitoring of blood units (Increase/Decrease logic).
* **Search Filters:** Quick search by Donor ID or Blood Group.
* **Security:** Secure Login system for administrators.

## 🛠️ Technical Stack
* **Frontend:** WinForms (C#)
* **Backend:** C# (.NET)
* **Database:** Microsoft SQL Server
* **Tools:** Visual Studio 2022

## ⚙️ Installation & Setup
1. Clone the repository: `git clone [Your-Repo-Link]`
2. Open the `BloodProject.slnx` file in Visual Studio.
3. Update the Connection String in `function.cs` to match your local SQL Server instance.
4. Run the SQL script (if provided) to create the `bloodbank` database.
5. Press **F5** to Build and Run.
